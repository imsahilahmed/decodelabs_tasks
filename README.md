# 📂 Projects

## 🤖 1. Custom Chatbot Web

A full-stack AI chatbot that maintains conversational context using an **in-memory history array** and a **sliding window algorithm**, enabling coherent multi-turn conversations during a user session.

### Highlights

* Session-based conversation memory
* Sliding Window (FIFO) context management
* Groq Llama 3.3 70B integration
* Flask backend with interactive web interface
* Chat history management and reset functionality

**Tech Stack**

* Python
* Flask
* HTML
* CSS
* JavaScript
* Groq API
* Llama 3.3 70B
